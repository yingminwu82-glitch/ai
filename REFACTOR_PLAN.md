# 实际可行方案（修正版）

## 数据情况
- 14 款 PNG 都是 896x1200 RGB（"美甲成品展示图"）
- 不是"标准甲片贴图"（带背景、姿势各异）
- 25 张手图也是 896x1200（标准评测图）

## 方案调整

### 旧方案（错位）
- LLM 框选指甲 → cv2 合成米色 → 贴到指甲位置

### 新方案（按需求 + 现实）
- **"指甲检测与分割"**：用 LLM 框选（接受 4/10 精度）
- **"白模"阶段**：先用 cv2 画白色椭圆/矩形（验证定位对不对）
- **"贴图"阶段**：从款式图里裁出"指甲区域"（8 个里选 1 个，因为同款式都涂一样），透视变换到用户指甲位置
- **alpha 混合**：白模/真实贴图都用 mask + GaussianBlur 边缘羽化

### 流程调整
- 前端：14 款式 gallery → 用户点款式 → 上传手图 → 试戴
- 替代品："自动推荐"按钮（不选款式时调 LLM 推荐）
- Chatbot 区域：选款 + 问问题
